# 🎮 五子棋网页游戏

一个功能完整的五子棋网页游戏，支持双人本地对战和AI对战模式。

## ✨ 功能特性

### 🎯 游戏模式
- **双人 PVP 模式**：两人在同一设备上轮流下棋
- **AI 对战模式**：挑战不同难度的AI对手

### 🤖 AI 难度等级
1. **简单** - AI随机下棋，偶尔犯错
2. **中等** - AI会进行基本的进攻和防守
3. **困难** - AI使用评估函数选择最佳位置
4. **绝密** - AI使用Minimax算法，几乎无敌

### ⚙️ 游戏设置
- **棋盘大小**：15×15、13×13、9×9、5×5
- **交换棋子**：随时交换黑白棋子颜色
- **重新开始**：随时重置游戏

### 🎨 界面特色
- 美观的棋盘和棋子设计
- 实时游戏状态显示
- 棋子计数统计
- 胜利连线高亮显示
- 响应式设计，支持移动设备

## 🚀 快速开始

## 🎮 游戏玩法

### 基本规则
1. 黑棋先行，双方轮流在棋盘交叉点上下棋
2. 先在横、竖、斜任意方向连成五子者获胜
3. 棋盘下满未分胜负则为平局

### 操作指南
- **下棋**：点击棋盘交叉点放置棋子
- **切换模式**：点击"模式"按钮切换双人/AI对战
- **交换棋子**：点击"交换棋子颜色"按钮
- **重新开始**：点击"重新开始"按钮

### 快捷键
- **R** - 重新开始游戏
- **M** - 切换游戏模式
- **S** - 交换棋子颜色

## 🧠 AI 算法说明

### 简单难度
- 随机选择空位下棋

### 中等难度
- 优先防守对方的四连威胁
- 否则随机下棋

### 困难难度
- 使用评估函数计算每个位置的得分
- 考虑进攻和防守的平衡
- 选择得分最高的位置

### 绝密难度
- 使用Minimax算法搜索2层深度
- Alpha-Beta剪枝优化性能
- 综合评估棋盘局势

## 🔧 自定义开发

### 修改棋盘大小
在 `game.js` 中修改 `boardSize` 属性：
```javascript
this.boardSize = 15; // 修改为 9, 13, 19 等
```

### 添加新的AI难度
在 `getAIMove()` 方法中添加新的难度逻辑。

### 修改样式
所有样式都在 `style.css` 中，可以自定义：
- 棋盘颜色
- 棋子样式
- 界面布局
